SkinsRestorer allows you to edit the current skin you have on.
Learn how to edit and upload your modified skin.

## Editing and uploading your skin

<Steps>
  <Step>
    ### Run the command

    Run the command `/skin edit` in the chat. This will give you a link to the skin editor.

  </Step>

  <Step>
    ### Edit the skin

    Use the skin editor to make changes to your skin. Once you are done, click on save the skin.

    ![Save edited skin](/assets/edit/save.png)

  </Step>

  <Step>
    ### Save the skin image

    Go to the left side and right click on the skin template shown in 2D.
    Right click the image and click on "save image as"

    ![Save image](/assets/edit/save-as.png)

  </Step>

  <Step>
    ### Upload using the /upload page

    Visit the [/upload](/upload) page on skinsrestorer.net and upload your saved skin PNG.
    The page will generate a ready-to-use `/skin url` command for you.

    See the [Skin URL Upload guide](/docs/features/skin-url) for detailed instructions.

  </Step>

  <Step>
    ### Apply your skin

    Copy the generated command and paste it in game or console to apply your edited skin.
    Make sure the URL stays wrapped in quotes.

  </Step>
</Steps>
