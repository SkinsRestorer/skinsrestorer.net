#### Disable or turn off TLauncher skins

1. Open TLauncher
2. Click on the `Accounts` button
3. Click on the `Configure account` selection
4. Uncheck the `Use skins TLauncher` checkbox
5. Click on the `Yes` button
6. You can now close the `Accounts` window

## SKlauncher

SKlauncher has its own skin system, which overrides server-side skin plugins such as SkinsRestorer.
To use SkinsRestorer skins, disable the SKlauncher skin system by enabling Compatibility Mode.

### How to fix

1. Open SKlauncher
2. Go to the `Installations Manager`
3. Edit the installation you use to join the server
4. Enable `Compatibility Mode`
5. Save the installation and launch Minecraft again

Compatibility Mode is configured separately for each installation.
Repeat these steps for every installation you want to use with SkinsRestorer.

For more information, see the
[official SKlauncher documentation](https://docs.skmedix.pl/faq/launcher-related#can-i-use-clients-mods-resource-packs-shaders-etc-in-sklauncher).

## Crystal Launcher

Crystal Launcher has its own skin system, which may override skins provided by SkinsRestorer.

### How to fix

1. Open **Launcher Settings**.
2. Select **Advanced**.
3. Enable **Disable Crystal Launcher skin system**.
4. Restart Minecraft.

## Legacy Launcher / TL Legacy

Legacy Launcher can apply Ely.by skins to offline accounts, overriding skins provided by SkinsRestorer.

### How to fix

1. Open the **Account Manager**.
2. Select the Free/offline account you use to play.
3. Edit the account.
4. Disable **Use Ely.by skins**.
5. Save the account and restart Minecraft.

When using an Ely.by account rather than an offline account, switch to a normal Free/offline account first.

## HMCL

HMCL can use its own skin system for offline accounts, which may override skins provided by SkinsRestorer.

### How to fix

1. Open the **Account List**.
2. Select your offline account.
3. Click **Upload/Edit Skin**.
4. Select **Default**.
5. Confirm the change and restart Minecraft.

Do not select Steve, Alex, a local skin file, LittleSkin, or Blessing Skin, since these options enable HMCL's offline skin integration.

## PCL2 / Plain Craft Launcher 2

PCL2's custom offline skin option uses a resource pack that replaces player skins locally.

### How to fix

1. Open **Settings**.
2. Open the **Launch** settings.
3. Find **Offline Skin**.
4. Click **Reset Skin**, or change the skin type from **Custom** to **Random**.
5. Restart Minecraft.
